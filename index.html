Installation Protocol

Deploy in Minutes

Get TchueKam Agent running locally on your hardware. Zero telemetry. Full control over your AI environment.

01
Prerequisites

Ensure your system has Python 3.10+ and Git installed. TchueKam works natively on macOS, Linux, and Windows.

python --version
02
Global Install

Install the core agent infrastructure directly from our repository or via pip. This will install the CLI tool.

pip install tchuekam-agent
03
Configuration

Set up your API keys. TchueKam supports DeepSeek, OpenRouter, Anthropic, and local models via Ollama.

tchuekam config set provider deepseek
04
Initiate Agent

Launch the agent interface. From here, you can assign goals, monitor tasks, and review system operations.

tchuekam start
